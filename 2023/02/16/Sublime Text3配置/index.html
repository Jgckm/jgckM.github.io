点击 Package Control: Install Package，需要等待一会，他会再次弹出搜索框

安装完成会弹出成功窗口

如果没弹出搜索框，在 Shift+Ctrl+P 输入 install 等一会就会出来了。

之后就是在搜索框里输入Chinese，选择下拉框中的ChineseLocalizations
